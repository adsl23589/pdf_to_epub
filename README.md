# 📄 PDF 轉 EPUB 智能轉換器 (PDF to EPUB Smart Converter)

這是一個專為「學術論文（尤其是人文學科）」設計的輕量級轉換工具。

學術論文的 PDF 往往包含雙欄排版、跨行斷字（Hyphenation）、複雜的頁首頁尾，直接丟進閱讀器通常會變成一團亂碼。這個小工具透過智能解析版面，幫你把 PDF 轉換成排版乾淨、流暢，且保留原本圖表的 EPUB 電子書格式，讓你在電子閱讀器上擁有最舒適的閱讀體驗！

✨ **安心保證**：本專案完全依賴如 `PyMuPDF`、`EbookLib` 等成熟的歐美開源套件，無任何中資背景軟體疑慮，請安心使用。

---

## 🌟 主要功能 (Features)

- **🖼️ 直覺拖曳介面**：不需要輸入複雜的指令，把 PDF 拖進視窗就能自動轉換。
- **📖 雙欄排版解析**：自動判斷左欄與右欄，按照正確的閱讀順序重新排列文字。
- **✂️ 智能去除頁首/頁尾**：自動過濾掉每頁邊緣干擾閱讀的期數、頁碼。
- **🧩 跨行斷字修復**：遇到行尾的連字號（例如 `infor- mation`），會自動查英文字典並無縫合併。
- **📊 圖表保留**：自動裁切 PDF 中的圖片並安插回 EPUB 中對應的段落。

## 🛠️ 安裝與啟動步驟 (新手友善指南)

如果你是第一次執行 Python 程式，請別擔心，跟著以下步驟一步步做即可！

### 第一步：確認你的電腦有安裝 Python
這支程式需要 Python 才能運作。
1. 請前往 [Python 官方網站](https://www.python.org/downloads/) 下載最新版本並安裝。
2. ⚠️ **重要提醒（Windows 用戶）**：安裝時，請務必勾選畫面底部的 **「Add python.exe to PATH」**，然後才點擊 Install Now。

### 第二步：下載這個專案
1. 點擊這個 GitHub 頁面右上角的綠色按鈕 **「<> Code」**。
2. 選擇 **「Download ZIP」**。
3. 將下載下來的 ZIP 檔案解壓縮，你會看到裡面有一個名為 `pdf_to_epub.py` 的檔案。

### 第三步：安裝必要的開源套件
我們需要安裝一些讓程式能夠解析 PDF 和字典的擴充包。
1. 打開你的**終端機 (Terminal)**（Mac 用戶）或 **命令提示字元 (cmd)**（Windows 用戶）。
2. 將以下這行指令複製，貼上到終端機後，按下 `Enter` 鍵：

   ```bash
   pip install PyMuPDF EbookLib pyspellchecker tkinterdnd2
```

3. 等待螢幕上的文字跑完，這代表套件已經順利安裝完畢。

### 第四步：啟動轉換器
1. 在終端機中，輸入 `cd` （注意 `cd` 後面有一個空格）。

2. 將你剛剛解壓縮的資料夾「拖曳」到終端機視窗裡，然後按下 `Enter`。這樣終端機就會切換到這個資料夾的目錄下。

3. 接著，輸入以下指令來啟動程式並按下 `Enter`：

```Bash
python pdf_to_epub.py
```

4. 成功啟動！這時你的畫面上會跳出一個小小的灰色視窗。

## 💡 如何使用 (Usage)
1. 準備好你想轉換的學術論文 PDF 檔。

2. 用滑鼠按住該 PDF 檔，直接**拖曳**到剛剛彈出的灰色小視窗中放開。

3. 視窗會顯示「轉換中，請稍候...」。

4. 幾秒鐘後，如果顯示「轉換成功！」，你就可以在原本 PDF 所在的同一個資料夾裡，找到熱騰騰剛轉好的 EPUB 檔案囉！把它傳進你的電子閱讀器開始享受閱讀吧！

## 📝 常見問題 (FAQ)
**Q: 為什麼有些文字還是會連在一起或斷行怪怪的？**
A: 因為 PDF 是一種「畫布」而不是單純的文字檔，有些極端特殊的排版（例如非常複雜的註腳混在正文中間）可能會讓機器誤判。目前版本主要針對人文學科常見的雙欄文字做最佳化。

**Q: 我可以轉換充滿複雜數學公式的論文嗎？**
A: 目前版本為了確保純文字的流暢度，遇到複雜的數學公式可能會出現亂碼或遺漏。如果您主要閱讀資工、物理等富含公式的論文，建議直接在平板上閱讀原始 PDF 檔。
